# Sum of First N Numbers - (using recursion)

## Code

```javascript
// sum of first n numbers
// using recursion

const num = 3;

const sum = (n) => {
  if (n < 0) return 0;
  return n + sum(n - 1);
};

console.log(sum(num));
```

## Overall Algorithm Logic

```mermaid
graph TD
    A["🟢 START<br/>n=3"] --> B{"n < 0?"}
    B -->|NO| C["return n + sum(n-1)<br/>3 + sum(2)"]
    C --> D["Call sum(2)"]
    D --> E{"2 < 0?"}
    E -->|NO| F["return 2 + sum(1)<br/>2 + sum(1)"]
    F --> G["Call sum(1)"]
    G --> H["Call sum(0)"]
    H --> I{"0 < 0?"}
    I -->|NO| J["return 0 + sum(-1)<br/>0 + sum(-1)"]
    J --> K["Call sum(-1)"]
    K --> L{"-1 < 0?"}
    L -->|YES| M["return 0<br/>Base case reached"]
    M --> N["Unwind stack<br/>sum(-1) = 0"]
    N --> O["sum(0) = 0 + 0 = 0"]
    O --> P["sum(1) = 1 + 0 = 1"]
    P --> Q["sum(2) = 2 + 1 = 3"]
    Q --> R["sum(3) = 3 + 3 = 6"]
    R --> S["🔴 Return 6<br/>Final result"]
```

## Example Walkthrough

For n=3:

- sum(3) = 3 + sum(2)
- sum(2) = 2 + sum(1)
- sum(1) = 1 + sum(0)
- sum(0) = 0 + sum(-1)
- sum(-1) = 0 (base case)
- Unwinding: 0 + 0 = 0, 1 + 0 = 1, 2 + 1 = 3, 3 + 3 = 6

**Result:** 6

## Complexity Analysis

- **Time Complexity:** O(n) - Each call makes one recursive call until n=0
- **Space Complexity:** O(n) - Recursion stack depth is n
- **Note:** This is not the most efficient way (iterative would be O(1) space), but demonstrates recursion
